# buckitlist example

## Dependencies
```
# babel
$ yarn add @babel/core @babel/preset-env -D

# babel polyfill
$ yarn add core-js

# eslint
$ yarn add eslint eslint-plugin-import eslint-config-airbnb-base -D

# webpack
$ yarn add webpack webpack-cli -D

# webpack loaders, plugins
$ yarn add babel-loader -D
$ yarn add html-webpack-plugin -D
$ yarn add css-loader sass-loader sass mini-css-extract-plugin -D
$ yarn add postcss-loader postcss-import postcss-preset-env cssnano -D
$ yarn add webpack-dev-server -D

# The sass-loader compress the sass in production, so do not need to install following dependencies.
$ yarn add yarn add terser-webpack-plugin optimize-css-assets-webpack-plugin -D
```
